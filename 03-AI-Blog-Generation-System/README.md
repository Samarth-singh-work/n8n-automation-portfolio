# 🤖 AI Blog Generation System

An AI-powered n8n workflow that automatically generates high-quality blog articles using OpenRouter AI and saves them as Google Docs.

---

## 🚀 Features

- AI-powered blog generation
- Topic and tone selection
- Adjustable word count
- SEO-friendly content creation
- Automatically creates a Google Document
- Inserts the generated blog into the document
- Sends the completed blog via email
- Built using n8n + OpenRouter + Google Docs

---

## 🛠 Tech Stack

- n8n
- OpenRouter API
- Google Docs API
- SMTP Email
- AI (Gemma 4 31B)

---

## 📋 Workflow

1. User submits a topic, tone, and word count.
2. AI generates a complete blog article.
3. A new Google Document is created.
4. The generated article is inserted into the document.
5. A notification email containing the generated blog is sent.
